# 猴子吃桃问题

猴子第一天摘下若干个桃子，当即吃了一半，还不过瘾，又多吃了一个。第二天早上又将剩下的桃子吃掉一半，又多吃了一个。以后每天早上都吃了前一天剩下的一半零一个。到第10天早上想再吃时，就只剩一个桃子了。编程求第一天共摘了多少桃子。

 

```
#include <stdio.h>

void main() {
  int i, s = 1;
  for(i = 0; i < 9; i++) { //因为到第十天还剩一个，所以循环9次
      s = (s + 1) * 2; //倒序求出每天的数量
  }

   printf("The first day:%d\n", s); //输出结果
}





94
46
22
10
4
1
```
